# UI/UX Redesign Backlog

## Summary

Backlog นี้แตกจากแผนรีดีไซน์ `budg69` ให้พร้อมเอาไปทำต่อเป็นรอบงานได้เลย โดยยึด 3 decisions ที่ล็อกไว้แล้ว:

- ใช้ `Unified Shell`
- ให้ `Support Budget` เป็น section ใน shell เดียวกัน
- ใช้ `Dense List/Table` เป็น working view หลัก

## Backlog

### Epic 1: Unified App Shell
- [ ] ออกแบบ app shell กลางสำหรับ `Index.html`
- [ ] กำหนด layout หลัก: sidebar, top command bar, content pane
- [ ] นิยาม section title และ active state ของแต่ละ section
- [ ] รองรับ desktop collapsible sidebar
- [ ] รองรับ mobile off-canvas sidebar
- [ ] ย้าย action หลักใน header ให้เหลือเฉพาะที่ควรอยู่ระดับ global
- [ ] กำหนด state model สำหรับ section navigation โดยไม่เปลี่ยน backend contract

**Acceptance**
- [ ] โหลดหน้าแล้วเห็น shell เดียวที่ใช้ได้ทั้งระบบ
- [ ] เปลี่ยน section ได้โดยไม่ต้องซ่อน/ย้าย DOM แบบ ad hoc
- [ ] mobile ใช้งาน nav ได้จริง
